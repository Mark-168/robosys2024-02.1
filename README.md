# range.py
![test](https://github.com/Mark-168/robosys2024-02.1/actions/workflows/test.yml/badge.svg)

受け取った値が
40より大きく60未満の場合は8,
40以下なら0
それ以外なら9の値を返します。
これらの返す値はJIS60062のカラーコードに順ずる物である。
